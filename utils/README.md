# AI Fact Checker

An AI-powered fact-checking application that analyzes PDF and DOCX documents, extracts factual claims, verifies them using live web search, and generates structured verdicts.

## Features

- PDF and DOCX support
- AI claim extraction
- Live web verification
- Structured verdict table
- False claim detection
- Outdated information detection

## Tech Stack

- Streamlit
- Groq API
- Tavily API
- Python

## Run Locally

Install dependencies:

pip install -r requirements.txt

Run app:

streamlit run app.py
